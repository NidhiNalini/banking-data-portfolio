# Banking Data Portfolio

Three connected data engineering projects built on synthetic 
banking transaction data, progressing from raw ETL through 
customer segmentation to an executive risk dashboard.

## Projects

### 01 — Transaction ETL Pipeline
Extracts, validates, and loads 500 banking transactions with 
FINTRAC-aware flagging and a full audit trail.

### 02 — Customer Segmentation
RFM scoring pipeline — 50 accounts segmented into HIGH_VALUE, 
NEW_CUSTOMER, AT_RISK, and DORMANT categories.

### 03 — Credit Risk Dashboard
Five-chart executive dashboard answering real banking risk and 
retention questions. 52% AT_RISK threshold warning included.
